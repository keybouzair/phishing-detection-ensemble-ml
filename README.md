# Phishing Website Detection using Ensemble Machine Learning

## Overview
This project focuses on detecting phishing websites using a hybrid approach combining:
- Machine Learning models (RF, SVM, KNN, NB)
- Ensemble techniques
- Rule-based analysis
- API-based real-time verification

## Problem
Traditional blacklist systems fail to detect zero-day phishing attacks.

## Solution
A multi-layer detection system:
- Feature extraction from URLs
- ML-based classification
- Ensemble decision making
- API validation (Google Safe Browsing)

## Tech Stack
- Python
- Scikit-learn
- Pandas, NumPy
- Flask (if used)

## Results
- Accuracy: ~91–95% (depending on configuration)
- Reduced false positives using ensemble approach
